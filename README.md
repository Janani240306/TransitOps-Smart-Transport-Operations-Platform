Important Note
This project is developed using Firebase Authentication and Cloud Firestore.
For security reasons, the Firebase project configuration used during development has been removed from this repository.

To Run the Project
Create your own Firebase project.
Enable the required Firebase services (Authentication and Firestore).
Replace the Firebase configuration with your own Firebase project credentials.
Run the application.
